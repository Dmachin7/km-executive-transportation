# Logo Placeholder

Place your logo file here as `logo.png` (or `logo.svg`).

- Recommended: SVG or high-res PNG with transparent background
- Optimal dimensions: ~280×112px (will be displayed at ~140×56px)
- File path: `/public/assets/logo/logo.png`
- Referenced in code as URL: `/assets/logo/logo.png`

The Navbar and Footer both include a text fallback ("KM EXECUTIVE") that
displays automatically until this file is present.
