# Vehicle Photos

Drop your vehicle images here. The site references these exact filenames:

| File            | Used in       | Recommended size    |
|-----------------|---------------|---------------------|
| `hero.jpg`      | Hero section background | 1920×1080px min |
| `suburban.jpg`  | About section image     | 800×1000px min  |

All images should be high-quality JPG or WebP.
Next.js will automatically optimize them for different screen sizes.

URL path in browser: `/assets/vehicle/filename.jpg`
